commoncpp
=========

Common C++ classes used in other projects.

### Dependencies
* CMake
* Boost (log, filesystem, system)
* Openssl (for SHA256)
* GraphicsMagic C++

###### Ubuntu 15.04 :
* cmake
* libboost-log-dev
* libboost-filesystem-dev
* libboost-system-dev
* libssl-dev
* libgraphicsmagick++1-dev
